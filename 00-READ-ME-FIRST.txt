✨ WHAT IS FEATURELAB?

An autonomous AI agent that:
  ✓ Loads your dataset
  ✓ Profiles all features (observe)
  ✓ Proposes transformations (hypothesize)
  ✓ Applies transformations (execute)
  ✓ Evaluates features (evaluate)
  ✓ Decides keep/discard (decide)
  ✓ Creates interactions (cross-features)
  ✓ Compares results (baseline vs engineered)

Result: Transform raw data into high-signal features automatically.

Tech Stack:
  Frontend: Next.js 14 + React 18 + Tailwind + Animations
  Backend: FastAPI + Python 3.11
  Database: Neon PostgreSQL
  AI/ML: Gemini API + Pinecone + scikit-learn + LightGBM
  Deploy: Vercel (frontend) + Render (backend)

🆘 SOMETHING WENT WRONG?

Common fixes:

Q: Build command not found?
A: Make sure you're in the directory with featurelab-build.prompt

Q: Port 3000 or 8000 already in use?
A: Kill existing processes or use different ports

Q: Version conflicts?
A: Run fix-all-versions.prompt first, then featurelab-build.prompt

Q: Database connection error?
A: Check DATABASE_URL in backend/.env file
