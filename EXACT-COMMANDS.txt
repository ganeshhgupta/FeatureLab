================================================================================
TROUBLESHOOTING
================================================================================

Build fails:
  claude --file featurelab-build.prompt --execute-all --verbose

Versions mismatch:
  claude --file fix-all-versions.prompt --execute-all

Backend won't start:
  cd backend && python -c "from main import app; print('✓')"

Frontend build fails:
  cd frontend && rm -rf node_modules && npm install && npm run build

Database error:
  psql postgresql://...neondb... -c "SELECT 1"

API returns 500:
  cd backend && python -c "from database import init_db, engine; init_db()"
